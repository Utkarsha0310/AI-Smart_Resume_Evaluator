AI-Driven Smart Resume Evaluator

This project delivers a next-generation AI-powered resume evaluation system built on Replit using FastAPI for the backend and TailwindCSS + Chart.js for the frontend. It is designed to outperform traditional Applicant Tracking Systems (ATS) by providing deep resume analysis, actionable recommendations, and professional reports that help candidates reach industry standards.

The app uses the Kaggle Resume Dataset
, which includes Resume.csv and over 2,400 categorized resumes, as a benchmark. Users upload PDF resumes, which are parsed using PyMuPDF for resilient text extraction. The content is then analyzed with spaCy, scikit-learn, and sentence-transformers, evaluating five key factors: formatting, grammar, keyword strength, readability, and ATS parse-ability. Each resume receives a 0–100 composite score, with high scorers (≥80) celebrated through a confetti animation and animated score display.

Beyond scoring, the system compares user resumes with top-performing exemplars from the dataset using clustering and embeddings. It highlights missing skills, weak phrasing, and structural gaps, then generates clear, prioritized improvement suggestions. The app also predicts relevant job roles aligned with the candidate’s profile, helping users discover career paths where they are most competitive.

The UI is modern, responsive, and visually engaging, built in a navy blue theme (#0A192F, #1F4068, #162447) with smooth animations and a light/dark mode toggle. Users navigate across four main pages:

index.html for uploading resumes

results.html for displaying ATS scores and detailed analysis

improve.html for actionable recommendations

visualization.html for advanced insights and analytics

The visualization dashboard features 15+ interactive Chart.js charts, displaying ATS score breakdowns, skill gaps, keyword density, grammar errors, readability levels, action verb usage, similarity to exemplar clusters, and improvement projections.

One of the most powerful features is the downloadable resume report. Generated using ReportLab, the report is produced in a perfectly formatted multi-page PDF with professional page setup. It not only summarizes the analysis but also includes personalized recommendations tailored to the specific resume uploaded live. Each report outlines key gaps, role suggestions, comparison scores, and step-by-step improvements, giving users a polished document they can use for career growth.

To balance speed and depth, the app supports a Heavy Mode toggle, letting users choose between fast evaluation with lightweight models (en_core_web_md, MiniLM) or deeper analysis with transformer models (en_core_web_trf, mpnet).

For performance, models are cached in /data/.models, exemplar clusters are precomputed, and FastAPI async routes ensure responsiveness. Built-in fallback strategies guarantee zero runtime errors, making the app stable and fully compatible with Replit—without relying on paid APIs.

The project maintains a clear structure: /modules for NLP pipelines, /templates for HTML pages, /static for assets and animations, /uploads for user files, and /data for datasets and models. run_all.py initializes pipelines and launches the server, while requirements.txt, .replit, and replit.nix ensure seamless setup.

In summary, the AI-Driven Smart Resume Evaluator combines state-of-the-art NLP, exemplar comparisons, interactive visual dashboards, and downloadable PDF reports with personalized insights. It goes far beyond an ATS, offering users not just a score but a clear improvement roadmap—making it an impressive, jaw-dropping tool for recruiters, professionals, students, and project evaluators alike.
